Title: Apex Legends Fanart
Genre: Digital Drawing
Work by: Yanmin Zhou
Date: Mar 30, 2023
Software: Procreate, Krita

This is a collection of Apex Legends fanart. I would consider this collection as an exploration of my style, and also a drawing practice. The style is inspired by a Japanese anime named "Panties and Stockings with Garterbelt".

The project started as a passion project. It consists of both static drawings and simple animations. This is a record of my self-learning process in the field of drawing.

The reason I chose the topic is that Apex Legends has a huge diversity of character designs that may challenge me.

The original work was made on iPad with Procreate. Some works have been redone with Krita on PC.

This is a portoflio website created by Yanmin Zhou with HTML and CSS. Works presented in this website are created by Yanmin Zhou. Some works may contain elements from the internet. Please note that this is not a public website and this is not an exhibition. Many works are made for academic purposes or personal interest.
